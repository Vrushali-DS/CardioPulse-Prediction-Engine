# ❤️ CardioPulse Prediction Engine
CardioPulse Prediction Engine is a production-inspired healthcare analytics platform developed to demonstrate the application of Machine Learning in clinical decision support systems.
The application processes patient demographic and medical information, applies trained classification models, and generates real-time risk predictions through an intuitive Streamlit interface. It also provides diagnostic visualizations and adjustable clinical thresholds to simulate real-world medical decision-making.
This project showcases the complete machine learning lifecycle—from data preprocessing and model development to deployment and interactive visualization.
# Key Features
✅ Real-Time Heart Disease Risk Prediction
✅ Machine Learning-Based Clinical Decision Support
✅ Interactive Patient Data Collection
✅ Adjustable Clinical Classification Threshold
✅ Multiple Predictive Models Comparison
✅ Feature Importance Visualization
✅ Patient vs High-Risk Cohort Analysis
✅ Responsive Streamlit Dashboard
# Objectives
Predict heart disease risk
Compare ML models
Visualize patient data
Assist clinical decision-making
# Dataset
The project uses the Heart Disease Dataset, which contains patient demographic information, medical history, and clinical measurements used for predicting the presence of heart disease.
Input Features
Age
Sex
Chest Pain Type
Resting Blood Pressure
Serum Cholesterol
Fasting Blood Sugar
Resting ECG
Maximum Heart Rate
Exercise-Induced Angina
ST Depression (Oldpeak)
Slope of ST Segment
Number of Major Vessels
Thalassemia
Target Variable
0 → Low Risk / No Heart Disease
1 → High Risk / Heart Disease
